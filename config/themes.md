# 収集テーマ

収集対象のテーマ。`## ` で始まる行が **1つの独立した収集グループ** になり、
その下の `- ` で始まる行がそのグループのテーマとして読み込まれる。

グループごとに独立して Web 検索するため、検索枠の食い合いによる取りこぼしが起きない。
**拡張・削除はグループ（`##` ブロック）単位で行える**：新テーマ系統を足すなら `##` を1つ追加し、
不要なら `##` ブロックごと消すだけでよい。人間が編集してよい。
